## Правила работы с персонажами

- Новый персонаж или вариация имени → сначала `define` в `game/characters.rpy`, затем строка в реестре выше.
- Вариации имени (ласковые формы, прозвища) — отдельные `Character(_("…"), kind=<база>)`: наследуют все свойства базового персонажа, меняется только отображаемое имя.
- Отображаемые имена — только через `_("…")`; переводятся через `translate <язык> strings` (подробно — README, «Мультиязычность»).
- В логике и сейвах имена персонажей как строки не использовать — только переменные/ключи.
