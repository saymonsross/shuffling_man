# Правило: README.md — индекс документации

- `README.md` — индексный файл документации проекта. Раздел «Документация» в нём обязан содержать актуальные ссылки на ВСЕ файлы в `docs/`.
- При любом добавлении, переименовании или удалении файла в `docs/` — в том же изменении обновить список ссылок в разделе «Документация» README.md.
- Ссылки — относительные (`docs/<файл>.md`), с коротким описанием содержимого файла.
- Не дублировать содержимое документов в README — только ссылки и однострочные описания.
