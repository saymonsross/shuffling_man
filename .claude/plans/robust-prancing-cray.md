# План: пролог, сцена 1 — начало письма и анимация взятия карандаша

## Контекст

Продолжаем наполнять `game/0_prologue/scene1.rpy` (лейбл `prologue_s1`). Сейчас сцена
заканчивается после мини-игры «навести порядок»: кросфейд в плоский композит
`prologue_note_center` (стол, бумага, карандаш, два смятых комка — **без рук**), три
реплики Марины и `return`.

Нужно **дописать следующий бит**: Марина заставляет себя начать писать. Для этого
«ровный» кадр записки надо собрать не из плоского jpg, а из фона и отдельных объектов
(`bg + бумага + карандаш`) — чтобы затем проиграть **автоматическую** (без действия
игрока) анимацию взятия карандаша: правая рука входит в кадр движением, тянется к
карандашу, затем сменяется позой письма. В момент смены на позу `prologue_hand_right
write` (она **уже держит карандаш** — проверено по ассету) табличный карандаш синхронно
убирается, чтобы карандаш «остался» только в руке.

Решения, согласованные с автором:
- **Ровный кадр**: заменить плоский `scene prologue_note_center` в конце сцены (стр. 213)
  на живую сборку из частей. Весь хвост (3 старых реплики + 3 новых + 1) и анимация идут
  поверх одного собранного кадра — без лишних свопов.
- **Анимация**: рука `move` въезжает движением (ease-подъезд + проявление) к карандашу,
  затем переключение на `write` с одновременным скрытием табличного карандаша.

Правки только в одном файле: **`game/0_prologue/scene1.rpy`**. Сценарный текст
неприкосновенен (`.claude/rules/scenario_text_immutable.md`) — новые реплики вставляются
дословно, существующие не трогаем.

## Ключевые факты (из разведки)

- Образы объявляются вручную (`images_auto()` не вызывается) — новый образ нужно
  объявить явно.
- `prologue_hand_right write.png` (798×954) **не объявлен** в проекте; ассет содержит руку
  с карандашом в позе письма.
- Позы рук объявлены как **отдельные теги** (`prologue_hand_right`, `prologue_hand_right_move`),
  чтобы сосуществовать и кросфейдиться по alpha. Новую позу объявляем так же —
  `prologue_hand_right_write` (отдельный тег), а не атрибутом.
- Композит `prologue_note center.jpg` = `prologue_note bg.jpg` + бумага + карандаш.
  Позиции измерены diff'ом композита и фона (1920×1080):
  - бумага: центр ≈ **(990, 455)**, без поворота
  - карандаш: центр ≈ **(1345, 432)**, вертикально, без поворота
- Смятые комки запечены в `prologue_note_bg` → сборка `bg + бумага + карандаш` совпадает
  с композитом.
- Готовые паттерны для переиспользования (в этом же файле): `note_hand_pos(pos_xy)`
  (статичная постановка руки, стр. 102), `prologue_dissolve`/`Dissolve(...)`, сборка кадра
  несколькими `show` под один `with` (стр. 189–195). 7dots `$ pause(...)` — «жёсткая»
  (несбиваемая кликом) пауза для автопроигрывания; при необходимости — `$ dismiss_off()` /
  `$ dismiss_on()` вокруг анимации.

## Изменения в `game/0_prologue/scene1.rpy`

### 1. Объявить образ позы письма
Рядом с другими руками (после стр. 34):
```renpy
image prologue_hand_right_write = "images/0_prologue/prologue_hand_right write.png"
```

### 2. Константы «ровного» кадра и анимации взятия
Рядом с блоком `NOTE_*` (около стр. 63). Позиции бумаги/карандаша измерены по композиту;
позиции рук — плейсхолдеры под финальную подгонку по месту (как и существующие `NOTE_*`):
```renpy
## Записка "в порядке" — те же объекты, что в беспорядке, но ровно (без поворота).
## Позиции бумаги/карандаша измерены по prologue_note center.jpg.
define NOTE_PAPER_NEAT_POS = (990, 455)
define NOTE_PENCIL_NEAT_POS = (1345, 432)

## Взятие карандаша (плейсхолдеры — подогнать по месту в игре).
define NOTE_HAND_REACH_FROM = (1450, 980)  # рука move стартует за нижним-правым краем
define NOTE_HAND_REACH_TO = (1250, 620)    # рука move дотянулась до карандаша
define NOTE_HAND_WRITE_POS = (1150, 700)   # поза письма: остриё над бумагой
```

### 3. Трансформы: ровная постановка + въезд руки
Рядом с существующими трансформами (около стр. 104). `note_hand_pos` для позы письма
переиспользуем как есть; добавляем:
```renpy
## Ровная постановка бумаги/карандаша (без поворота).
transform note_paper_neat:
    anchor (0.5, 0.5)
    pos NOTE_PAPER_NEAT_POS

transform note_pencil_neat:
    anchor (0.5, 0.5)
    pos NOTE_PENCIL_NEAT_POS

## Рука тянется к карандашу: въезжает движением из-за кадра с проявлением.
transform note_hand_reach_in(from_xy, to_xy, t=0.9):
    anchor (0.5, 0.5)
    pos from_xy
    alpha 0.0
    parallel:
        ease t alpha 1.0
    parallel:
        ease t pos to_xy
```

### 4. Переписать хвост сцены (стр. 210–222)
Заменить блок «камера/плоский композит + 3 реплики + return» на сборку из частей +
существующие реплики + новые реплики + анимацию:
```renpy
    ## Порядок наведён — кросфейд в ровную записку, собранную из частей (bg +
    ## бумага + карандаш, без рук), чтобы дальше анимировать карандаш. Камера
    ## сброшена — кадр статичен.
    camera
    scene prologue_note_bg
    show prologue_note_paper at note_paper_neat
    show prologue_note_pencil at note_pencil_neat
    with Dissolve(0.6)

    "Теперь всё ровно. Можно начинать."

    "Я здесь после нервного срыва, что разрушил мою и без того распадавшуюся на части жизнь и подорванное здоровье."

    "Это письмо... должно помочь мне пережить произошедшее."

    "Долго я не находила в себе сил, чтобы записать случившееся. Ушло много попыток."

    "Не было сил вспоминать: меня трясло, рвало, руки непроизвольно тянулись закрыть лицо."

    "Обо всём случившемся невыносимо думать."

    "Но я должна излить наружу то, что пожирает меня изнутри."

    ## Взятие карандаша (без участия игрока). Рука тянется движением к карандашу,
    ## затем сменяется позой письма; в тот же момент табличный карандаш убирается —
    ## карандаш "остаётся" только в руке (поза write уже держит его).
    $ dismiss_off()
    show prologue_hand_right_move at note_hand_reach_in(NOTE_HAND_REACH_FROM, NOTE_HAND_REACH_TO)
    $ pause(0.9)

    show prologue_hand_right_write at note_hand_pos(NOTE_HAND_WRITE_POS)
    hide prologue_hand_right_move
    hide prologue_note_pencil
    with Dissolve(0.3)
    $ pause(0.4)
    $ dismiss_on()

    return
```

Примечания:
- Единый `with Dissolve(0.3)` относится ко всем трём предыдущим `show`/`hide` → поза
  письма проявляется, а поза `move` и табличный карандаш гаснут **синхронно** — ровно
  «убрать карандаш в момент смены на write».
- Используем «сырые» объекты `prologue_note_paper` / `prologue_note_pencil` (не
  ConditionSwitch-обёртки `note_paper`/`note_pencil` — подсветка тут не нужна).
- Рук в ровном кадре изначально нет (совпадает с композитом); правая рука входит только
  на анимации. Левая рука в этом бите не участвует.
- `image prologue_note_center` (стр. 18) после правки больше не используется в сцене —
  **оставляем** как эталон-референс для подгонки позиций; комментарий обновим.
